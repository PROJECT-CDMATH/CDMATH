subroutine mlnnln ( integer  fid,
integer  n,
integer  cret 
)

Cette routine permet la lecture du nombre de lien dans un fichier MED.

Paramètres:
fid Identificateur du fichier.
n Nombre de lien dans le fichier.
cret retour négatif en cas d'erreur, Zéro sinon.
Voir également:
MEDnLink

Cette routine permet la lecture du nombre de lien dans un fichier MED. Dans MED, un champ et un maillage peuvent être dans deux fichiers différents. Dans le fichier contenant le champ, il est nécessaire de créer un lien spécifiant le chemin d'accès et le nom du fichier MED contenant le maillage. Le lien porte le nom du maillage.
